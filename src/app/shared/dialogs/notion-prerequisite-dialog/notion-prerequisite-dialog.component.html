Generate the Media Library Notion Database

Notion Prerequisites

In order to correctly use the Shitshup App, you must create the Media Library Database. No worries, we'll do it automatically for you, but we want to explain you the ins and outs.

Just make sure that the root block id in your Notion Config is set. It is needed in order to create the media library within it.

What's that all about ?

The Media Library Database will regroup all the details of your tracks, and it's the base for all the workflow. Thanks to Notion and all the other tools used in the Shitshup App, you will be able to access all the informations about the BPM, Keys and Time signatures,... While also being able to add and modify other data that only you, as an artist, can provide ! Like the common use-cases, the noticeable layers, mood, personal rating,...
