Welcome to Shitshup UI !

Requirements

This app downloads YouTube video from a specific YouTube playlist from your account, and converts them to .mp3 files. If you want to use this app at its fullest, you'll need :

  • YouTube channel (mandatory)
  • Linux or Windows OS : used to perform musical analysis via Essentia (optionnal)
  • Notion account : used to save the result in a Notion database (optionnal)
  • Dropbox account : used to embed audio files to the corresponding Notion database rows (optionnal)

How to use

First, authenticate yourself with your YouTube channel and follow the next instructions. This app will automate the creation of the playlist that Shitshup UI will use for the downloads.

Next, you'll have access to different sections :

  • The ability to process one or multiple videos at once. Either from a unique link or ID, or from the playlist.
  • Tasks : indications about the progress of your processes
  • Warnings : will tell you what's going wrong
  • Camelot Wheel conversion : helper tool used to better match your audios

Special Thanks

This app uses MTG's Essentia library : a wonderful library that performs easy-to-use audio analysis !

This app uses two libraries for Chords conversions : regorxxx's Camelot-Wheel-Notation & jackbittiner's camelot-wheel !
